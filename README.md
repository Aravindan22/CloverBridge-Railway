# CloverBridge-Railway

Simple django application to check the seat position
- Input - Seat number (0 < seat_number < 73)
- Output - Seat position


## Run Locally

Clone the project

```bash
  git clone https://github.com/Aravindan22/CloverBridge-Railway.git
```

Go to the project directory

```bash
  cd cloverbridge
```

Install dependencies

```bash
  python -m pip install requirements.txt
```

Start the local server

- Local server will run on port 8000
```bash
  python manag.py runserver
```
